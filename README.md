# Slopaganda

A Next.js website for the Slopaganda research concept, focused on Phase 1: Authority & Thought Leadership.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.

## Project Structure

- `src/app/page.tsx` — main landing page content and Phase 1 messaging
- `src/app/layout.tsx` — site metadata and root layout
- `src/app/globals.css` — global theme and typography

## Next Steps

After Phase 1, you can extend this site with:

- an education section for explainers and guides
- an interactive slopaganda tracker or detection tools
- media and press asset pages
- community resources or consultation services
